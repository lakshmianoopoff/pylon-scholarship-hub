# 📘 Pylon – Scholarship Finder Website

## 🌟 Overview

**Pylon** is a responsive web application designed to help students find scholarships that match their academic profile, financial background, and career interests. It simplifies the process of discovering relevant funding opportunities through an easy-to-use filtering system.

---

## 🎯 Features

* 🔍 **Scholarship Filtering System**

  * Filters based on age, income, GPA, field of study, and residency
* 🎓 **Multiple Scholarship Categories**

  * Government
  * Private
  * International
* ⚡ **Dynamic Results Display**

  * Shows matching scholarships instantly
* 🎨 **Modern UI Design**

  * Clean layout with animations and transitions
* 📱 **Responsive Design**

  * Works on desktop and mobile devices
* 📩 **Contact Form**

  * Allows users to send queries or feedback

## 🚀 How It Works

1. User fills out the eligibility form:

   * Age
   * Income
   * GPA
   * Field of Study
   * Residency Status

2. The system:

   * Matches user input with predefined scholarship data
   * Filters eligible scholarships

3. Results:

   * Displays matching scholarships with details
   * Provides direct application links

---

## 🧠 Core Logic

* Scholarships are stored as an array of objects in `script.js`
* Filtering is done using:

  * Age limit check
  * Income threshold
  * GPA requirement
  * Field match
  * Residency match
